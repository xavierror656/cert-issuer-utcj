# Branding UTCJ

## Origen de recursos

- logo oficial localizado en `https://www.utcj.edu.mx/wp-content/uploads/2025/09/logo_utcj.png`
- manual institucional enlazado desde el sitio publico UTCJ como `manual-de-imagen-2025.pdf`

## Propuesta visual

- base institucional en verde profundo y teal tecnologico
- soporte en blanco y gris grafito para sobriedad academica
- acento dorado suave para autenticidad y valor verificable
- composicion contemporanea, limpia y con foco en portabilidad profesional del logro academico

## Paleta

- verde institucional: `#0F6A52`
- verde profundo: `#0A4C3B`
- azul/teal tecnologico: `#0F3E4A`
- gris grafito: `#1F2937`
- blanco: `#FFFFFF`
- oro suave: `#B88A3B`

## Composicion del certificado

- encabezado con identidad UTCJ
- nombre del receptor con maxima jerarquia visual
- titulo de la microcredencial como elemento protagonista
- bloque de competencias acreditadas
- metadata de emision, horas y folio unico
- QR hacia el JSON verificable
- espacio visual para logo UTCJ y referencia criptografica
